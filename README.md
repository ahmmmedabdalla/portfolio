# Elevare Consulting Website

This project is a professional one-page consulting website built with plain HTML, CSS, and JavaScript.

## Website Purpose

The website presents a consulting business called **Elevare Consulting** and includes:

- Home / landing section
- Professional navbar
- Services section
- About Us section
- Our Clients section
- Contact Us section
- WhatsApp contact button
- Arabic / English language switcher
- Photo-based premium hero background
- Enhanced footer with quick links and contact area

## Files

- `index.html` - main website structure and content
- `styles.css` - visual design, layout, responsiveness, and branding
- `script.js` - mobile menu behavior, language switching, and contact form interaction

## Run Instructions

1. Open the project folder.
2. Double-click `index.html` to open the site in your browser.

If you want a local development server instead, you can use:

```powershell
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Customization Instructions

- Replace the business name if you want a different company name.

This is a static website, so there is no installation step or framework build process required.

You can later upgrade it to:

- React
